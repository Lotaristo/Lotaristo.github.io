Image Captioning

В рамках проекта по генерации описаний изображений (Image Captioning) использован датасет Flickr8k для обучения модели, которая создает текстовые описания для изображений. Модель основывается на архитектуре Seq2Seq с предобученной моделью InceptionV3 для извлечения признаков изображений и LSTM для генерации текста. Проект продемонстрировал использование методов обработки изображений и естественного языка для решения задачи генерации описаний

Генерация изображений с использованием DCGAN

В этом проекте использован датасет MNIST Fashion для генерации изображений одежды с помощью модели DCGAN (Deep Convolutional Generative Adversarial Network). Модель состоит из генератора и дискриминатора, обученных на изображениях одежды, с целью создания реалистичных новых образов. Проект продемонстрировал применение генеративных состязательных сетей для синтетической генерации изображений на основе данных о реальных объектах

Сегментация изображений легких с использованием U-net

В проекте по сегментации легких использована архитектура U-Net++ для обработки медицинских изображений, с целью выделения и точной сегментации областей легких на рентгеновских снимках. Модель была обучена на датасете с изображениями здоровых легких, что позволило добиться высокой точности в выделении структуры легких.
